## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Mihir9702/Portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the dev server on localhost
   ```sh
   npm run dev
   ```
